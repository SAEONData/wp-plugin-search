== Description ==

### SAEON Data Search: Shortcode for searching the SAEON ODP

Use the shortcode to insert a search field on any page or widget location. The search runs results from the SAEON ODP, a comprehensive South African environmetal database.

#### Control your input style

Use attributes to control the input styling with custom CSS

#### Control your results listing

Use shortcode attributes to control:

* Using an iframe or new tab for results
* Use a results shortcode to display results on the page other than below the search field
* Show or hide search filters in results listing

== Installation ==
Starting with SAEON Data Search consists of just two steps: installing and inserting the shortcode.

### INSTALL SAEON Data Search FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select ‘Add New’;
1. Search for ‘SAEON Data Search’;
1. Activate SAEON Data Search from your Plugins page;

### INSTALL SAEON Data Search MANUALLY

1. Upload the ‘saeon-data-search’ folder to the /wp-content/plugins/ directory;
1. Activate the SAEON Data Search plugin through the ‘Plugins’ menu in WordPress;

### AFTER ACTIVATION

1. Navigate to the plugin ‘Data Search’ in your admin menu;
1. Copy the shortcode and add any attributes you would like;
1. Paste shortcode into the page and You’re done!

== Frequently Asked Questions ==

= Can I use this data for commercial purposes? =

No, the data contained in the SAEON ODP is for the public good and cannot be used for profit.

= How often is SAEON Data Search updated? =

SAEON Data Search is updated roughly every month.
